# AI Interviewer

A fully local, offline AI-powered mock interview application. Upload your resume, get tailored interview questions generated by a local LLM, answer them by voice or text, and receive a scored report with per-question feedback and emotion analysis. No API keys or internet connection required after the initial model download.

## Features

Resume parsing supports both PDF upload and pasted plain text. A local Phi-3.5-mini LLM generates 5 personalized interview questions based on your resume content, mixing technical and behavioral prompts. You can answer questions by recording your voice (transcribed locally via Whisper) or by typing directly. Each question is read aloud automatically using the system TTS engine.

During the interview, your webcam is shown in a video call style panel with a live emotion badge overlay. A ViT face-expression model samples your camera every 2 seconds and records your dominant emotion per question. When you finish, the same LLM evaluates all your answers alongside the emotion data and produces a detailed scored report with overall feedback, per-question scores and written feedback, and a scrollable card list of the full session.

## Requirements

Python 3.10 or higher, a webcam and microphone, and around 5 GB of disk space for model weights (downloaded automatically on first run). A CUDA-capable GPU is optional but significantly speeds up LLM inference.

## Installation

Clone the repository and navigate into it:

```bash
git clone https://github.com/your-username/ai-interviewer.git
cd ai-interviewer
```

Create and activate a virtual environment:

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

For GPU support, replace the plain torch install with the CUDA build:

```bash
pip install torch --index-url https://download.pytorch.org/whl/cu128
```

## Running the App

```bash
python -m src.gui.app
```

Model weights are downloaded from Hugging Face on the first run and cached locally. Subsequent launches load from cache.

## Usage

Start on the home screen and click Start Interview. On the resume input screen, upload a PDF or paste your resume text, then click Generate Questions. This takes 30 to 60 seconds on first run while the LLM loads.

On the interview screen, questions appear in an overlay at the bottom of the interviewer panel and are read aloud automatically. Use the left and right arrow buttons to navigate between questions. Click Record Answer to speak your response, then Stop Recording to transcribe it, or type directly into the text box. The webcam feed is shown in the bottom left with your current emotion displayed as an overlay badge. Click Finish Interview when you are done.

The results screen scores your answers while you wait, then displays the full report. Scroll through the per-question cards or click Start Over to return to the resume input screen.

## Tools and Libraries

| Purpose | Library or Model |
|---|---|
| GUI framework | Pygame |
| PDF parsing | pdfplumber |
| LLM (questions and scoring) | microsoft/Phi-3.5-mini-instruct via Hugging Face Transformers |
| Speech to text | openai/whisper-base via Transformers |
| Text to speech | pyttsx3 (system TTS, fully offline) |
| Audio capture | sounddevice |
| Webcam capture | OpenCV (opencv-python) |
| Emotion detection | trpakov/vit-face-expression via Transformers and OpenCV Haar cascade |
| ML acceleration | PyTorch and accelerate |
| Testing | pytest |
